Deep Learning for Actuaries

Lecture slides from UNSW’s ACTL3143/5111

Author

Dr Patrick Laub

These are the lecture slides from my recent “Deep Learning for Actuaries” courses (coded ACTL3143 & ACTL5111) at UNSW.

This repository can be used to see:

  1. what topics I covered in these courses,
  2. an example for making UNSW-themed reveal.js slides with Quarto.

These slides are not intended to be used to learn deep learning from scratch. For that, you need to attend the lectures & complete the assessment.

Note

Most lectures start with a ‘preamble’ of feedback & admin things that are probably not too interesting for a general audience; press ‘o’ to go to toggle the overview mode, and use the arrow keys to quickly skip past these to the main content. E.g., I would recommend starting at this slide in Lecture 2; Lecture 1 was mainly an introduction to the course and to the Python language.
